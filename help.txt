Метод DotProduct получает на вход два вектора с размерностью 3, где каждый элемент вектора типа long long. 
Возвращает скалярное произведение, целое значение типа long long.

Метод VectorProduct получает на вход два вектора с размерностью 3, где каждый элемент вектора типа long long.
Возвращает вектор с размерностью 3, где каждый элемент типа long long.

Метод MultiVectorProduct получает на вход три вектора с размерностью 3, где каждый вектора элемент типа long long.
Возвращает смешанное произведение, целое значение типа long long.

Метод IsCollinear получает на вход два вектора с размерностью 3, где каждый элемент вектора типа long long.
Возвращает логическое значение типа bool - коллениарный ли исходные два вектора или нет.

Метод IsComplanar получает на вход три вектора с размерностью 3, где каждый элемент вектора типа long long.
Возвращает логическое значение типа bool - компаланарные ли исходные три вектора или нет.
